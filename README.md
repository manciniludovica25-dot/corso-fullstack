# Corso Fullstack — Progetti

Questo repository contiene più mini-progetti, ognuno nella propria cartella.


Al momento i progetti **non sono separati in sottocartelle**: sono presenti nella root del repository e si distinguono dal nome del progetto/file.

## Contenuti
- **Backend**: esercizi in Java / Spring Boot
- **Frontend**: esercizi in HTML / CSS / JavaScript
- **Full Stack**: progetti completi (frontend + backend + database MySQL)

## Note
Ogni progetto può includere istruzioni di esecuzione nei commenti o in un file dedicato, quando necessario.

## Convenzione versioni HTML (progetti statici)

Per i progetti in HTML:

- **`index.html`** = **ultima versione** (quella da considerare “corrente” e da deployare)
- **File che iniziano con `P`** = **versioni precedenti**, rinominate con la data di creazione

Formato consigliato:
- `PYYYYMMDD_nomeProgetto.html` (es. `P20260423_articolo.html`)
- se ci sono più versioni nello stesso giorno: `PYYYYMMDD_nomeProgetto1.html` (es.`P20260423_articolo2.html` )
